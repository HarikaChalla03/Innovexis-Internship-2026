# Local Food Wastage Management System

## Introduction
The Local Food Wastage Management System is a data-driven web application developed to reduce food wastage and improve food distribution efficiency. It helps connect food providers and receivers through a structured database and interactive dashboard. The system allows users to manage food listings, monitor claims, and analyze donation and wastage trends.

## Objective
The main objective of this project is to reduce local food wastage by building a centralized platform where surplus food data can be stored, tracked, and analyzed. It also aims to connect food providers with receivers and support better decision-making using SQL queries and dashboard insights.

## Features
- Dashboard with key metrics such as providers, receivers, food listings, claims, and total quantity
- SQL-based analysis for business insights
- Food listings filter by city, food type, meal type, and provider type
- CRUD operations for adding, updating, and deleting records
- MySQL database integration
- Interactive Streamlit user interface
- Claim tracking and status analysis
- Data-driven insights for food availability and utilization

## Tech Stack
- Frontend/UI: Streamlit
- Backend: Python
- Database: MySQL
- Libraries: Pandas, SQLAlchemy
- Tool: Visual Studio
